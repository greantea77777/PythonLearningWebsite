1. 객체지향 프로그래밍(OOP)

class Dog:
    pass

객체지향 프로그래밍은 현실 세계의 객체를 코드로 표현하는 방법입니다.

Python의 클래스, 객체, 상속, 다형성 등이 OOP의 핵심 개념입니다.

2. 클래스(Class)

class Student:

    def __init__(self, name):
        self.name = name

student = Student("Tom")

print(student.name)

클래스는 객체를 만들기 위한 설계도입니다.

객체마다 서로 다른 데이터를 가질 수 있습니다.

3. Flask 웹 개발

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello Flask"

Flask는 Python으로 웹사이트를 만들 수 있는 프레임워크입니다.

현재 여러분의 Python Learning Website도 Flask로 제작되었습니다.

4. 데이터베이스(SQLite)

import sqlite3

conn = sqlite3.connect("database.db")

cursor = conn.cursor()

cursor.execute(
    "SELECT * FROM users"
)

conn.close()

데이터베이스는 사용자 정보와 학습 데이터를 저장합니다.

이 프로젝트에서는 SQLite를 사용하여 회원가입, 로그인, XP를 관리합니다.

프로젝트 실전 예제

@app.route("/dashboard")
def dashboard():

    return render_template(
        "dashboard.html"
    )

Flask는 URL과 HTML 페이지를 연결하여 동적인 웹사이트를 만듭니다.
