Intermediate Python 학습

오늘 배울 내용

  • 함수(Function)
  • 리스트(List)
  • 딕셔너리(Dictionary)
  • 파일 입출력(File I/O)

1. 함수(Function)

def hello():
    print("Hello")

hello()

함수는 코드를 재사용하기 위해 만드는 기능입니다.

2. 리스트(List)

fruits = ["apple", "banana", "orange"]

print(fruits[0])

리스트는 여러 개의 데이터를 순서대로 저장합니다.

3. 딕셔너리(Dictionary)

student = {
    "name": "Tom",
    "age": 20
}

print(student["name"])

딕셔너리는 key와 value 형태로 데이터를 저장합니다.

4. 파일 입출력(File I/O)

with open("test.txt", "w") as f:
    f.write("Hello")

파일에 데이터를 저장하거나 읽어올 수 있습니다.

실습 예제

numbers = [10, 20, 30]

for n in numbers:
    print(n)

리스트와 반복문을 함께 사용하는 예제입니다.
