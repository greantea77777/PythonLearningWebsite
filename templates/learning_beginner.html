Python 기초 학습

오늘 배울 내용

  • print() 함수
  • input() 함수
  • 변수(variable)
  • if 조건문
  • for 반복문

1. print() 함수

print("Hello World")

print() 함수는 화면에 내용을 출력하는 함수입니다.

2. input() 함수

name = input("이름을 입력하세요")

사용자로부터 입력을 받을 때 사용합니다.

3. 변수 (Variable)

name = "Python"
age = 20

변수는 데이터를 저장하는 공간입니다.

4. if 조건문

age = 20

if age >= 20:
    print("성인")

조건이 참일 때만 실행됩니다.

5. for 반복문

for i in range(5):
    print(i)

같은 작업을 여러 번 반복할 때 사용합니다.
